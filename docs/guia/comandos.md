# Comandos úteis

| Comando                       | Descrição                           |
|------------------------------|-------------------------------------|
| poetry run task lint         | Formata e valida o código           |
| poetry run task test         | Executa os testes                   |
| poetry run task coverage     | Relatório de cobertura dos testes   |
| poetry run task docs         | Inicia a documentação em localhost  |
