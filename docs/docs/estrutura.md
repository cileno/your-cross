# Estrutura do Projeto

```text
your-cross/
├── pyproject.toml
├── mkdocs.yml
├── src/
│   ├── config/         ← Projeto Django
│   └── your_cross/     ← Pacote do sistema
├── tests/              ← Testes automatizados
├── docs/               ← Documentação do projeto
└── .venv/              ← Ambiente virtual (via poetry)
```
